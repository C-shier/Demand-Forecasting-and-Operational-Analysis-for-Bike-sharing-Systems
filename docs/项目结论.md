# 项目结论

## 一、项目结果概览

本项目基于 2011-2012 年共享单车小时级租赁数据，完成了需求规律分析与小时级需求预测。结果显示：

1. 总租赁量为 3,292,679，小时平均租赁量为 189.46。
2. 注册用户贡献了 81.17% 的租赁量，平台核心场景以稳定通勤为主。
3. 需求高峰明显集中在工作日前后的早晚通勤时段，其中 17 点平均租赁量最高。
4. 随机森林模型在三类模型中表现最好，RMSE 为 105.2536，R2 为 0.7274。

## 二、核心业务发现

### 1. 时间规律清晰

- 7-8 点和 17-18 点形成明显双高峰，符合通勤型出行特征。
- 月度趋势存在季节性波动，说明运力安排不能只按平均值配置。

### 2. 天气与需求强相关

- 晴天/少云条件下平均租赁量最高，为 204.87。
- 恶劣天气下平均租赁量降至 74.33，需求会出现明显收缩。

### 3. 模型可以作为调度辅助

- 基线模型已经能说明时间和天气对需求的基础解释力。
- 随机森林进一步捕捉了非线性关系，更适合作为小时级需求预测方案。

## 三、可落地建议

1. 将工作日标签、小时维度和天气预报结合，形成补车排班规则。
2. 在 7-8 点和 17-18 点前完成重点站点补车，降低缺车率。
3. 恶劣天气前适当下调运力投放，减少空驶和闲置成本。
4. 对注册用户聚集站点优先保障供给，因为这部分用户代表更稳定的复购需求。
